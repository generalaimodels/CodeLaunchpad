HTML id & <iframe>: 10 Examples

Example 1: Basic id Usage

The id attribute assigns a unique identifier to an HTML element. It must be unique within the page. Useful for CSS, JavaScript, and anchor links.
<div id="uniqueSection">This div has a unique id.</div>
This div has a unique id.

Example 2: Using id for CSS Styling

You can target an element by its id in CSS using the #idName selector.
<style> #specialBox { background: #e0f7fa; color: #006064; padding: 1rem; border-radius: 6px; } </style> <div id="specialBox">Styled using id selector in CSS.</div>
Styled using id selector in CSS.

Example 3: Using id for JavaScript Access

id allows direct access to elements in JavaScript using getElementById.
<div id="jsTarget">Click the button to change my text.</div> <button onclick="document.getElementById('jsTarget').textContent = 'Text changed!';" class="btn">Change Text</button>
Click the button to change my text.

Example 4: Exception - Duplicate id (Invalid)

Exception: id values must be unique. Duplicates cause unpredictable behavior in CSS/JS.
<!-- Invalid: --> <div id="duplicate">First div</div> <div id="duplicate">Second div (invalid)</div>
First div
Second div (invalid)
Do NOT use the same id more than once!

Example 5: Anchor Links with id

id can be used as an anchor target for in-page navigation.
<a href="#anchorTarget">Go to Section</a> ... <div id="anchorTarget">You have jumped to this section!</div>
Go to Section
You have jumped to this section!

Example 6: Basic <iframe> Embedding

The <iframe> element embeds another HTML page within the current page. The src attribute specifies the URL.
<iframe src="https://www.example.com" width="300" height="120"></iframe>

Example 7: <iframe> with id and JavaScript Access

Assign an id to an <iframe> for dynamic manipulation (e.g., changing src).
<iframe id="myIframe" src="https://www.wikipedia.org" width="400" height="220"></iframe> <button onclick="document.getElementById('myIframe').src='https://www.example.com';">Change Iframe Source</button>

Example 8: Responsive <iframe> with CSS

Make iframes responsive using CSS. The aspect-ratio property ensures the iframe scales with the container.
<div class="iframe-container"> <iframe src="https://www.wikipedia.org" class="responsive-iframe"></iframe> </div>

Example 9: Iframe Security - sandbox and allow

Use sandbox to restrict iframe capabilities for security. allow can grant specific permissions.
<iframe src="https://www.example.com" width="300" height="120" sandbox title="Sandboxed Iframe"></iframe>

Example 10: Exception - Iframe Loading Errors & Fallback

If the iframe cannot load the content (e.g., due to network or permissions), the fallback text is shown.
<iframe src="https://notarealwebsite.abc" width="300" height="120"> Sorry, the content could not be loaded. </iframe>
If the iframe fails, the fallback text is displayed (try disabling your internet or using a broken URL).
