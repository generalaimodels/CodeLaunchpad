HTML Page Title & Tables: 10 Detailed Examples

This file demonstrates the use of HTML page titles and tables in various scenarios, covering syntax, attributes, accessibility, and best practices. Each example is explained in detail for clarity.

1. Basic Page Title

The <title> element defines the title of the HTML document, shown in the browser tab and used by search engines. 

<title>HTML Page Title & Tables: 10 Examples</title>
  • Place <title> inside the <head> section.
  • Keep titles concise and descriptive for SEO and usability.

2. Dynamic Page Title (with JavaScript)

You can change the page title dynamically using JavaScript. 

<script>
    document.title = "New Page Title";
</script>
  • Useful for single-page applications or dynamic content.
  • Try waiting 3 seconds to see the title change in your browser tab.

3. Basic Table Structure

The simplest table uses <table>, <tr> (table row), <th> (header cell), and <td> (data cell). 

<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>Alice</td>
        <td>24</td>
    </tr>
</table>
Name Age
Alice 24
  • Always use <th> for header cells for accessibility.

4. Table with <caption> and Summary

Use <caption> to provide a title or summary for the table. 

<table>
    <caption>Student Grades</caption>
    ...
</table>
Student Grades
Student Grade
Bob A
Carol B+
  • <caption> improves accessibility and context for screen readers.

5. Table with <thead>, <tbody>, <tfoot>

Use these elements to semantically group table sections. 

<table>
    <thead>
        <tr><th>Product</th><th>Price</th></tr>
    </thead>
    <tbody>
        <tr><td>Book</td><td>$10</td></tr>
        <tr><td>Pen</td><td>$2</td></tr>
    </tbody>
    <tfoot>
        <tr><td>Total</td><td>$12</td></tr>
    </tfoot>
</table>
ProductPrice
Book$10
Pen$2
Total$12
  • Improves accessibility and enables sticky headers/footers in some browsers.

6. Table with rowspan and colspan

Use rowspan and colspan to merge cells vertically or horizontally. 

<table>
    <tr>
        <th rowspan="2">Name</th>
        <th colspan="2">Scores</th>
    </tr>
    <tr>
        <th>Math</th>
        <th>Science</th>
    </tr>
    <tr>
        <td>Eve</td>
        <td>95</td>
        <td>98</td>
    </tr>
</table>
Name Scores
Math Science
Eve 95 98
  • rowspan merges cells vertically, colspan merges horizontally.

7. Accessible Table with scope Attribute

Use scope="col" or scope="row" for <th> to improve accessibility for screen readers. 

<table>
    <tr>
        <th scope="col">City</th>
        <th scope="col">Country</th>
    </tr>
    <tr>
        <td>Paris</td>
        <td>France</td>
    </tr>
</table>
City Country
Paris France
  • Improves navigation for assistive technologies.

8. Table with Empty Cells and <td> vs <th>

Use <th> for headers, <td> for data. Empty cells are valid and sometimes useful. 

<table>
    <tr><th>Item</th><th>Value</th></tr>
    <tr><td>A</td><td>10</td></tr>
    <tr><td>B</td><td></td></tr>
</table>
ItemValue
A10
B
  • Empty <td> cells are allowed and can be styled as needed.

9. Table with Inline CSS Styling

You can style tables and cells using inline CSS or classes. 

<table style="border:2px solid #1976d2;">
    <tr>
        <th style="background:#bbdefb;">Header</th>
        <td style="color:#d84315;">Styled Cell</td>
    </tr>
</table>
Header Styled Cell
  • Use CSS classes for maintainability in larger projects.

10. Responsive Table (Horizontal Scroll)

For wide tables, wrap in a container with overflow-x: auto; to enable horizontal scrolling on small screens. 

<div style="overflow-x:auto;">
    <table>...</table>
</div>
Header 1Header 2Header 3Header 4Header 5
Data 1Data 2Data 3Data 4Data 5
  • Improves usability on mobile devices.

Additional Notes & Exceptions

  • Do not use tables for layout; use them only for tabular data.
  • Always provide table headers for accessibility.
  • Use <caption> for table titles, not as a replacement for <th>.
  • For complex tables, use scope, id, and headers attributes for accessibility.
  • Empty <td> cells can be filled with &nbsp; if you want to ensure a minimum cell height.
  • Page title is critical for SEO and user experience; keep it unique and relevant.
