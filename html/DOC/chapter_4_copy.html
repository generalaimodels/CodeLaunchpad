HTML Comments

Comments in HTML help document your code, explain logic, or temporarily disable code during development. They are invisible to users and do not affect the rendered page. 

<!-- This is a single-line HTML comment -->
<p>Visible content</p>

Visible content

<!--
  This is a multi-line comment.
  It can span several lines.
-->
    
<!-- <h2>This heading is commented out and will not show</h2> -->
    
<p>Hello <!-- This comment is inside a paragraph --> World!</p>

Hello World!

<!-- Incorrect: <!--- Too many dashes ---> or <! This is not a comment > -->
Exception: Comments must start with <!-- and end with -->. Do not use extra dashes or omit the exclamation mark.
