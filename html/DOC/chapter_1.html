HTML Introduction

HTML is the backbone of all web pages. It allows you to structure content, embed images, create links, and much more. Every website you visit is built using HTML at its core. 

<!DOCTYPE html>
<html lang="en">
  <head>
    <title>My First HTML Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is my first web page.</p>
  </body>
</html>
    
<HTML>
  <BODY>
    <H1>Case Insensitivity</H1>
  </BODY>
</HTML>
    
<html>
  <body>
    <h1>    Whitespace     </h1>
    <p>HTML ignores extra spaces.</p>
  </body>
</html>
    
<!-- This is a comment. It is not displayed in the browser. -->
    
<!-- HTML is a markup language, not a programming language. -->

HTML Editors

You can write HTML in any text editor (Notepad, Vim, Nano, etc.), but using a code editor improves productivity and reduces errors.

  • VS Code (Visual Studio Code): Free, powerful, extensible.
  • Sublime Text: Fast, lightweight, customizable.
  • Atom: Open-source, hackable.
  • Notepad++: Lightweight, Windows-only.
  • Online Editors: CodePen, JSFiddle, Repl.it, etc.
1. Open your editor (e.g., VS Code).
2. Write your HTML code.
3. Save the file with a .html extension, e.g., index.html.
4. Open the file in a web browser to view the result.

HTML Basic Structure

Every HTML document should start with a doctype declaration and have a root <html> element. The <head> contains meta-information, and the <body> contains the visible content. 

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>Basic HTML Structure</title>
  </head>
  <body>
    <h1>Welcome!</h1>
    <p>This is a basic HTML page.</p>
  </body>
</html>
    
<!DOCTYPE html>
<html><head><title>Minimal</title></head><body>Hi!</body></html>
    
<meta name="viewport" content="width=device-width, initial-scale=1.0">
    
<!-- This is a comment. Comments help explain code and are ignored by browsers. -->
    
<html>
  <body>
    <h1>Missing <head> and <!DOCTYPE></h1>
  </body>
</html>
Note: Browsers try to fix errors, but always use correct structure!

HTML Elements

An HTML element is defined by a start tag, some content, and an end tag. Some elements are self-closing (void elements). 

<p>This is a paragraph element.</p>
    
<div>
  <h2>Nested Example</h2>
  <p>A <strong>strong</strong> word inside a paragraph.</p>
</div>
    
<img src="logo.png" alt="Logo">
<br>
<hr>
    
<p></p>  <!-- An empty paragraph, renders as a blank line -->
    
<b><i>Bold and Italic</b></i>  <!-- Incorrect nesting -->
Browsers try to fix incorrect nesting, but always nest tags properly!

HTML Attributes

Attributes modify the behavior or appearance of elements. They are written as name="value" pairs inside the opening tag. 

<a href="https://www.example.com">Visit Example</a>
    
<img src="cat.jpg" alt="A cute cat">
    
<p style="color: blue; font-size: 18px;">Styled text</p>
    
<div id="main-content" class="container highlight">Content here</div>
    
<input type="checkbox" checked>

HTML Headings

Headings are used to define the structure and hierarchy of your content. Use them in order for accessibility and SEO. 

<h1>Main Title</h1>
<h2>Section Title</h2>
<h3>Subsection</h3>
<h4>Detail</h4>
<h5>Minor Detail</h5>
<h6>Smallest Heading</h6>
    
<h1>HTML Guide</h1>
  <h2>Introduction</h2>
  <h2>Basics</h2>
    <h3>Elements</h3>
    <h3>Attributes</h3>
  <h2>Advanced</h2>
    
<h1>Title</h1>
<h3>Subsection</h3>
Do not skip heading levels for accessibility and SEO. 
<h2 style="color: green;">Styled Heading</h2>
    
<h2>Heading</h2>
<b>Bold text, not a heading</b>
Use headings for structure, not just for bold text.

HTML Paragraphs

Paragraphs are used to group sentences and blocks of text. Each <p> starts on a new line. 

<p>This is a simple paragraph.</p>
    
<p>First paragraph.</p>
<p>Second paragraph.</p>
    
<p>
  This      text
  has    lots of
  spaces.
</p>
    
<p>Line one.<br>Line two.</p>
    
<p>Paragraph <p>Nested paragraph</p></p>
Do not nest <p> tags. Browsers will auto-close the first <p>.
