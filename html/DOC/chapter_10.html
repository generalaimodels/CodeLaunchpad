HTML <div> and Classes: 10 Examples

Example 1: Basic <div> Usage

The <div> element is a generic container for flow content. It has no semantic meaning by itself. Use it to group elements for styling or scripting.
<div>This is a simple div container.</div>
This is a simple div container.

Example 2: Applying a Single Class

Classes allow you to apply CSS styles to one or more elements. Here, we use class="red-box" to style the div.
<div class="red-box">This div has a red background.</div>
This div has a red background.

Example 3: Multiple Classes

You can assign multiple classes to a single element, separated by spaces. All styles from those classes are applied.
<div class="blue-box highlight">This div is blue and highlighted.</div>
This div is blue and highlighted.

Example 4: Grouping Content

<div> is often used to group related content, such as a card with a title and description.
<div class="card"> <h3>Card Title</h3> <p>This is a card description.</p> </div>

Card Title

This is a card description.

Example 5: Nesting <div> Elements

<div> elements can be nested to create complex layouts or groupings.
<div class="card"> <div class="highlight">Header</div> <div>Body content here.</div> </div>
Header
Body content here.

Example 6: Layout with Flexbox

<div> is commonly used for layout. Here, we use classes and Flexbox to create a responsive row of columns.
<div class="flex-row"> <div class="column">Column 1</div> <div class="column">Column 2</div> <div class="column">Column 3</div> </div>
Column 1
Column 2
Column 3

Example 7: Dynamic Class Assignment

Classes can be added, removed, or toggled using JavaScript for interactivity.
<div id="toggleDiv" class="card">Click the button to highlight me!</div> <button onclick="toggleHighlight()" class="btn">Toggle Highlight</button> <script> function toggleHighlight() { document.getElementById('toggleDiv').classList.toggle('highlight'); } </script>
Click the button to highlight me!

Example 8: Exception - Avoid
for Semantic Content

Best Practice: Use semantic elements (<header>, <nav>, <main>, etc.) for meaningful content. Use <div> only when no semantic element fits.
<!-- Bad: --> <div>This is a navigation bar</div> <!-- Good: --> <nav>This is a navigation bar</nav>
Bad:
This is a navigation bar
Good:
This is a navigation bar

Example 9: Responsive Design with Classes

Classes can be used to make <div> elements responsive. Resize the window to see the effect.
<div class="responsive card"> This card is responsive and centered. </div>
This card is responsive and centered.

Example 10: Multiple Classes for Theming

Multiple classes can be used to switch themes (e.g., dark/light mode) dynamically.
<div id="themeDiv" class="card theme-light">This div changes theme.</div> <button onclick="toggleTheme()" class="btn">Toggle Theme</button> <script> function toggleTheme() { const el = document.getElementById('themeDiv'); el.classList.toggle('theme-dark'); el.classList.toggle('theme-light'); } </script>
This div changes theme.
