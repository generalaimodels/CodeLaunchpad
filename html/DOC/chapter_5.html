HTML CSS: Introduction

CSS (Cascading Style Sheets) is the language for describing the look and formatting of HTML documents. It separates content (HTML) from presentation (CSS), making web development more efficient and maintainable.

Example 1: Inline CSS

<p style="color: red; font-weight: bold;">This is styled with inline CSS.</p>

Example 9: Responsive Design with Media Query

@media (max-width: 600px) {
  body { background: #e0f7fa; }
  section { padding: 0.5rem; }
}
Resize the browser window to see the background and padding change on small screens.
