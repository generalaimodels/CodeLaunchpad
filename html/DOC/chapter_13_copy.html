HTML Layout: 10 Examples

Example 1: Semantic Layout Structure

Use semantic HTML5 elements (<header>, <nav>, <main>, <aside>, <footer>) for clear, accessible layouts.
<header>Header</header> <nav>Navigation</nav> <main>Main Content</main> <aside>Sidebar</aside> <footer>Footer</footer>
Header
Navigation
Main Content
Sidebar
Footer

Example 2: Classic Layout with <div> and CSS

Before HTML5, layouts were often built with <div> and CSS classes.
<div class="header">Header</div> <div class="nav">Nav</div> <div class="row"> <div class="col sidebar">Sidebar</div> <div class="col-2 content">Content</div> </div> <div class="footer">Footer</div>

Nav

Example 3: Two-Column Layout with Flexbox

Flexbox is a modern, powerful way to create flexible layouts.
<div style="display:flex;"> <div style="flex:1;">Left Column</div> <div style="flex:2;">Right Column</div> </div>
Left Column
Right Column

Example 4: Grid Layout with CSS Grid

CSS Grid enables two-dimensional layouts with rows and columns.
<div style="display:grid; grid-template-columns:repeat(3,1fr); gap:1rem;"> <div>1</div> <div>2</div> <div>3</div> </div>
1
2
3

Example 5: Float-based Layout (Legacy)

Before Flexbox/Grid, floats were used for multi-column layouts. Not recommended for new projects.
<div class="float-container clearfix"> <div class="float-left">Left</div> <div class="float-right">Right</div> </div>
Left
Right

Example 6: Centering Content (Vertically & Horizontally)

Flexbox makes centering content easy both vertically and horizontally.
<div style="display:flex;justify-content:center;align-items:center;height:120px;"> Centered Content </div>
Centered Content

Example 7: Responsive Layout with Flexbox

Responsive layouts adapt to screen size. Flexbox with flex-wrap and media queries enables this.
<div class="responsive-layout"> <div>Box 1</div> <div>Box 2</div> <div>Box 3</div> </div>
Box 1
Box 2
Box 3

Example 8: Nested Layouts (Complex Structure)

Layouts can be nested for complex designs, such as a sidebar with its own navigation.
<div class="row"> <aside class="sidebar"> <nav>Sidebar Nav</nav> </aside> <main class="main"> <div class="content">Main Content</div> </main> </div>
Sidebar Nav
Main Content

Example 9: Exception - Table-based Layout (Obsolete)

Exception: Table-based layouts are obsolete and harm accessibility. Use only for tabular data, not page layout.
<!-- Bad Practice: --> <table> <tr><td>Sidebar</td><td>Content</td></tr> </table>
SidebarContent
Do NOT use tables for layout! Use semantic elements, Flexbox, or Grid instead.

Example 10: Layout with Fixed and Fluid Elements

Combine fixed-width and flexible elements for advanced layouts.
<div style="display:flex;"> <div style="width:200px;">Fixed Sidebar</div> <div style="flex:1;">Fluid Content</div> </div>
